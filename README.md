# mind-flow-frontend# 🧠 MindFlow AI Frontend

MindFlow AI is an AI-powered Second Brain platform that helps users capture, organize, and retrieve knowledge efficiently using natural language queries.

The frontend provides a modern, responsive user interface for interacting with personal knowledge bases, managing categories, and communicating with AI-powered search capabilities.

## 🎯 Problem Statement

Students and developers often struggle to manage large amounts of notes, documentation, and learning resources.

Finding relevant information later becomes time-consuming and inefficient.

---

## 💡 Solution

MindFlow AI acts as a personal knowledge assistant by:

- Organizing information into categories
- Storing knowledge securely
- Allowing users to query stored information using natural language
- Delivering AI-powered responses based on saved content

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/LakinsaniSaiGanesh8800/mind-flow-ai-frontend.git
```

### Navigate to Project

```bash
cd mind-flow-ai-frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Application will run on:

```text
http://localhost:5173
```
